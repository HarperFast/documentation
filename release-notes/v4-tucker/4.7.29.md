### HarperDB 4.7.29

4/28/2026

- Allow downgrading to a previous version (with confirmation prompt)
- Removed NATS install script
- Backported env-var config fixes from Harper: when `securePort` is set to the same value as the HTTP port, the HTTP port is now automatically nulled instead of failing validation; added `composeConfigFromEnv()` helper for inspecting merged config without modifying the config state file
- Improved certificate handling for replication: default certificate now includes replication key usages, and certificate scoring updated to heavily favor certs with specific usage types
