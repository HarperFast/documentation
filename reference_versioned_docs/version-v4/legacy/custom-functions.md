title: Custom Functions

<!-- Source: versioned_docs/version-4.1/custom-functions/index.md (primary) -->

Custom Functions were Harper's original mechanism for adding custom API endpoints and application logic to a Harper instance. They allowed developers to define Fastify-based HTTP routes that ran inside Harper with direct access to the database, and could be deployed across instances via Studio.

Custom Functions were superseded by the [Components](../components/overview.md) system introduced in v4.2.0. Components provide the same capabilities with a more robust architecture, better tooling, and support for extensions and plugins.

All users are encouraged to migrate Custom Functions to Components. See the [Components](../components/overview.md) documentation for the modern approach.
