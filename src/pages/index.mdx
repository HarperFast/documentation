# Harper Docs

Harper is an all-in-one backend that fuses a database, caching, application hosting, and messaging into a single system — eliminating the serialization overhead and network latency of traditional multi-service architectures.

:::info Join the Community

Get help from Harper's core team, collaborate with other developers, and stay current on platform updates. [Join our Discord →](https://harper.fast/discord)

:::
